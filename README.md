# Tropi Runner Tracker

A single-page tropical running dashboard that can be opened directly from `index.html` or hosted on any static site provider.

## Features

- Dashboard totals for distance, run count, average pace, and runs this week
- Run logger with date, distance, duration, and optional average heart rate
- Editable and deletable run history stored in browser localStorage
- Responsive tropical theme with an offline canvas progress chart

## Open the app

Open `index.html` in a browser, or publish the repository root with GitHub Pages, Replit static hosting, Netlify, Vercel, or any other static file host.
